üìå Repository Path Configuration

The Repository Path Env field tells the progress tracking system which environment variable contains the path to your local repository clone.

This allows the system to:

  • Access the git repository to analyze commits
  • Track PRs and development activity
  • Generate accurate progress reports

Example Configuration:

# In your .env file: DURABLE_CODE_PATH=/home/stevejackson/Projects/durable-code-test

The Docker container will mount this repository and use it for progress tracking.
